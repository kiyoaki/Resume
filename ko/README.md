**시니어 게임 엔지니어 / 테크니컬 리더**

대규모 모바일·콘솔 타이틀의 개발과 운영, 분산·다국적 팀의 기술 매니지먼트 경험을 보유하고 있습니다. Unreal/Unity 클라이언트 구현부터 고부하 트래픽을 견디는 ASP.NET Core와 클라우드 기반의 설계·운영까지 엔드 투 엔드로 주도합니다.
